JPA

JPA is only a specification, it is not an implementation.
JPA is a set of rules and guidelines.
JPA requires the support of interfaces and classes.
It supports polymorphism and inheritance.

Hibernate 

Hibernate is an implementation of JPA guidelines.
It helps in mapping java data type of SQL.
it is contributor of JPA.

 Hibenate performs an object relational mapping framework while JDBC is simply
